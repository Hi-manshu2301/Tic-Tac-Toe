# Tic Tac Toe Game

A modern Tic Tac Toe game built using HTML, CSS and JavaScript.

## Features

- Single Player Mode (AI)
- Multiplayer Mode
- Winner Detection
- Draw Detection
- Winning Animation
- Restart Game Functionality
- Responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript
